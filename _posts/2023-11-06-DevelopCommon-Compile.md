# 내용

## Compile 이란
>개발자가 프로그램을 위해 작성한 소스크드를 다른 프로그램이나 기계어로 바꾸는 과정

## Compile 언어란
- Compile 과정이 필요한 언어

### Compile 언어 종류
- Java
- C
- C++

## 스크립트 언어란
- 컴파일 과정 없이 기계어로  번역되는 즉시 동작
- 실제 작동시에는 컴파일 언어보다 느리다

### 스크립트 언어 종류
-  Javascript
- PHP
- Python

## Compile 타임 에러
> Compile 에러는 프로그램이 컴파일링 되는 과정에서 발생하는 에러
> 일반적으로 컴파일 에러는 IDE에서 확인이 가능

### Compile 에러 종류
- Syntax Error
- Type 체크 에러
- 파일 참조 오류
