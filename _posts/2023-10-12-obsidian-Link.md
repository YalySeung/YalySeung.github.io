## 링크 방법
  
| syntax                       | 설명                       |
| ---------------------------- |:-------------------------- |
| \[\[노트명\]\]               | 기본링크                   |
| \[\[노트명\#헤더명\]\]       | 노트의 헤더 링크           |  
  
| \[\[\#헤더명\]\]             | 현재 노트의 헤더 링크      |
| \[\[노트명#^특정문단라인\]\] | 특정 문단 링크             |  
  
| \[\[#^특정문단라인\]\]       | 현재 노트의 특정 문단 링크 |
| \!\[\[파일명\]\]             | 파일링크                   |
| \!\[\[노트명\]\]             | 노트 직접 링크             |
| \[\[노트명\|Alias]\]         | Alias 링크                 |
