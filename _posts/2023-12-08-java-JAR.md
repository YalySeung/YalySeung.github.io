> **JAR란?**  
>
> Java Archive의 약어로, 압축 파일 포맷중 하나이다. 여러개의 자바 클래스 파일, 리소스 및 메타데이터를 하나의 파일로 모은 소프트웨어 패키지 파일 포맷이다.
