## 정의
> **WAS 란?**  
>
> 웹 서버 + 웹 컨테이너
> 인터넷 상에서 HTTP를 통해 사용자 컴퓨터나 장치에 애플리케이션을 수행하는 미들웨어
> 동적 서버 콘텐츠를 제공 
{: .notice--info}  
  
## 기능
- 프로그램 실행 환경과 데이터베이스 접속 기능을 제공
- 여러 개의 트랜잭션을 관리
- 업무 처리 비즈니스 로직 수행
- Web Service 플랫폼 역할
