# 내용

## 정의
> **JPA란?**
>
> Java Persistence API
> 자바 객체를 관계형 데이터 베이스에 영속적으로 저장하고 조외할 수 있는 ORM 기술에 대한 표준 명세
{: .notice--info}

## 특징
- JPA는 인터페이스의 모음, 단순 명세
- 구현은 사용자가 직접 해야한다.

## 장점
- SQL 쿼리를 작성하지 않고, 객체를 통해 데이터베이스를 조작 가능
