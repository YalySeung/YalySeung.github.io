# 내용

## 커맨드

### 이동
- :startRow,endRow : 줄 선택
- gg : 글 처음으로 이동
- G : 글 끝으로 이동

### 명령
- i : insert mode
- d : delete
- c : change
- y : yank
- = : 소스정렬
- :%s/대상/변경 : replace all
- / : search
