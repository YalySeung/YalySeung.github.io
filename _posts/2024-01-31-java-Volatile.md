# 내용

## 정의
> **Volatile 이란?**
>
> java 변수를 Main Memory에 저장하는 키워드
{: .notice--info}

## 특징
- 변수값 Read 시점에 CPU 캐시된 값이 아닌 Main Memory에서 Read
- 변수값 Wirte 시점에 Main Memory 값 갱신
