> **JRE란?**  
>
> Java Runtime Environment의 약어로, 소프트웨어를 실행하도록 설계된 소프트웨어의 일부이다. 클래스 라이브러리, 클래스로더, 자바명령, JVM등 으로 구성된 되어있다.
