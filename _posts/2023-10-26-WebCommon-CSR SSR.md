# 내용

## CSR
- Client Side Rendering
- 첫 화면 생성 느림
- SPA
- HTML을 클라이언트에서 생성
- 새로고침 없이 부드러운 화면전환
- 서버 부하 비교적 적음
- 페이지를 나중에 그려나가는 방식때문에 검색 엔진에 노출이 잘 안됨

## SSR
- Server Side Rendering
- 첫 화면 생성 빠름
- MPA
- 화면 전환시, 새로고침 발생
- 서버 부하 비교적 많음
