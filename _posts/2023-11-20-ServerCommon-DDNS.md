# 내용

## 정의
> DDNS란
>Dynamic Domain Name System
>IP 주소가 변경될 때, DNS 레코드를 자동으로 업데이트 할 수 있는 서비스

## 기능
- 유동 IP를 고정 IP처럼 사용 가능
- IP 주소가 동적으로 할당 및 변경되는 경우 DNS 서버 레코드 업데이트
- 도메인 이름 관리가 더 쉽고 효율적
