# 내용

## 정의
> **JVM이란?**
>
> 자바 바이트 코드를 실행하는 Virtual Machine
{: .notice--info}

## 특징
- 플랫폼에 의존적(Window, Linux.. 운영체제별 JVM 존재)
- 컴파일된 바이트 코드는 어떤 JVM에서나 동작한다.
- 바이트 코드 읽기, 검증, 실행, 실행환경의 규격 제공 등의 역할을 한다.
