> **JVM이란?**  
>
> Java Virtual Machine의 약어로, 자바 바이트 코드를 실행하는 가상 머신을 의미한다. 
{: .notice--info}  

JVM은 **플랫폼에 의존적**(Window, Linux.. 운영체제별 JVM 존재) 이지만 컴파일된 **바이트 코드는 어떤 JVM에서나 동작**한다. JVM은 **바이트 코드 읽기**, **검증, 실행**, **실행환경의 규격 제공** 등의 역할을 한다.
