# 내용

## 정의
> **BDD란?**
>
> Behavior Driven Development
> TDD에서 파생된 개발 방법론으로 사용자 행위를 기반으로 사용자 시나리오를 구성하여 테스트 케이스를 작성하고 개발을 진행하는 방식
{: .notice--info}

## BDD 수행 과정
1. 요구사항 명세
2. 테스트 케이스 작성
3. 테스트 수행
4. 코드 구현
5. 테스트 확인
6. Refactoring
