# 내용

## 태그

#### 사용법

```html
<태그명>콘텐츠</태그명>
<태그명/>
```

#### 작성 유의사항
- 태그명은 소문자로 쓴다.
- 태그의 콘텐츠로 태그를 추가할 수 있다.

### 속성
- 태그에 부가 기능을 추가하기 위해 작성

```html
<태그명 속성명="속성값">콘텐츠</태그명>
<img src="dog.png" alt="강아지 이미지">
```
