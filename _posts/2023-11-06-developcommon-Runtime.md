## 정의
> **Runtime 이란?**  
>
> 컴파일 과정을 마친 컴퓨터 프로그램이 실행되고 있는 환경 또는 동작되는 동안의 시간 
{: .notice--info}  
  
## Runtime 환경 예시
- Web Browser
- Nodejs
  
## Runtime 에러
- 0으로 나누기 시도
- Null Exception
- OOM(Out Of Memory)
