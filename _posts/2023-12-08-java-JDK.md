# 내용

## 정의
> JDK란?
> Java Development Kit
> Java 애플리케이션을 만드는 데 사용되는 소프트웨어 개발 환경

## 특징
- Java 프로그램을 만드는 도구 + 이를 실행하는 도구([JRE](../../java/java-JRE)) => 모든 JDK 버전은 [JRE](../../java/java-JRE)와 함께 제공된다.
- javac(컴파일러), jar(아카이브), jdb(디버거), Javadoc(문서 생성기) 등을 포함한다.
